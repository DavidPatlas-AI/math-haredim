זוויות במשולש

סכום 180°, זווית חיצונית, סיווג משולשים — כללי הבסיס

החוק הראשון — סכום זוויות ⭐

α + β + γ = 180°

בכל משולש, סכום שלוש הזוויות הפנימיות הוא 180°. זה החוק הכי בסיסי בגיאומטריה.

זוויות: 50°, 60° → השלישית: 180° − 50° − 60° = 70°
משולש ישר-זווית: 90° + 35° + β = 180° → β = 55°
משולש שווה-שוקיים: זוויות בסיס 40° כל אחת → זווית ראש = 180° − 80° = 100°
💡 טיפ
אם ידועות שתי זוויות, השלישית היא תמיד 180° פחות סכום השתיים. לא צריך לחשוב — פשוט לחסר.

זווית חיצונית משפט

זווית חיצונית במשולש היא זווית שנוצרת כשמאריכים צלע אחת. היא שווה לסכום שתי הזוויות הפנימיות שאינן סמוכות לה.

δ = α + β
במשולש זוויות 40° ו-60° — הזווית החיצונית לזווית השלישית = 40° + 60° = 100°
בדיקה: זווית שלישית = 80°. החיצונית = 180° − 80° = 100° ✓
שתי דרכים לחישוב: זווית חיצונית = 180° − זווית פנימית סמוכה או = סכום שתי הפנימיות האחרות.

סיווג משולשים לפי זוויות סוגים

חד-זוויות
כל הזוויות < 90°
ישר-זווית
זווית אחת = 90°
קהה-זווית
זווית אחת > 90°
שווה-שוקיים
שתי זוויות בסיס שוות

תכונות נובעות שימושי

במשולש ישר-זווית: שתי הזוויות החדות משלימות ל-90° (α + β = 90°)
במשולש שווה-צלעות: כל זווית = 60°
במשולש שווה-שוקיים: זוויות הבסיס שוות. סימנו זווית בסיס = x → זווית ראש = 180° − 2x
זווית חיצונית במשולש שווה-צלעות = 120°

טעויות נפוצות להיזהר

1. שוכחים שסכום = 180°. קיבלתם 200°? כנראה טעות בחישוב או בזיהוי.

2. מבלבלים זווית חיצונית. חיצונית = 180° − פנימית סמוכה. לא 180° − פנימית אחרת!

3. במשולש ישר-זווית: החדות משלימות ל-90°, לא ל-180°.

4. חושבים שכל משולש עם שתי זוויות שוות הוא שווה-שוקיים. נכון — אבל גם ההפך: במשולש שווה-שוקיים, זוויות הבסיס שוות.
