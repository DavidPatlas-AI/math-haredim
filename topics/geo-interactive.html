📖 משפט פיתגורס — הסבר

במשולש ישר-זווית, הצלע מול הזווית הישרה נקראת יתר (c). שתי הצלעות האחרות הן ניצבים (a, b).

a² + b² = c²

שימוש:

  • חישוב יתר: c = √(a² + b²)
  • חישוב ניצב: a = √(c² − b²)

דוגמה: a=3, b=4 ⟹ c=√(9+16)=√25=5

📖 שטח טרפז — הסבר

טרפז הוא מרובע עם זוג אחד של צלעות מקבילות (בסיסים). הגובה הוא המרחק האנכי בין הבסיסים.

שטח = ½ × (a + b) × h
  • a = בסיס גדול, b = בסיס קטן
  • h = גובה (ניצב לשני הבסיסים)

דוגמה: a=6, b=4, h=3 ⟹ שטח = ½×10×3 = 15

📖 פונקציה ליניארית — הסבר

y = mx + b
  • m = שיפוע: m>0 עולה, m<0 יורד, m=0 אופקי
  • b = נקודת חיתוך עם ציר y: (0, b)
  • חיתוך עם ציר x: הצב y=0 ⟹ x = −b/m

שיפוע בין שתי נקודות: m = (y₂−y₁)/(x₂−x₁)

📖 משוואה ריבועית — הסבר

למשוואה ax²+bx+c=0, חשב תחילה את הדיסקרימיננט:

Δ = b² − 4ac
  • Δ > 0 — שני שורשים: x = (−b ± √Δ) / (2a)
  • Δ = 0 — שורש כפול: x = −b / (2a)
  • Δ < 0 — אין שורשים ממשיים

ראש הפרבולה: xᵥ = −b/(2a), yᵥ = f(xᵥ)
