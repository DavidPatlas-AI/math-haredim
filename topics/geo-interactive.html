📖 משפט פיתגורס — הסבר

במשולש ישר-זווית, הצלע שמול הזווית הישרה נקראת יתר (c). שתי הצלעות האחרות הן ניצבים (a ו-b).

a² + b² = c²

שימוש: מחפשים צלע חסרה:

  • יתר: c = √(a² + b²)
  • ניצב: a = √(c² − b²)

דוגמה קלאסית: a=3, b=4 ← c=√(9+16)=√25=5

📖 שטח טרפז — הסבר

טרפז הוא מרובע עם זוג אחד של צלעות מקבילות. הצלעות המקבילות הן הבסיסים (a ו-b), והמרחק ביניהן הוא הגובה (h).

שטח = ½ × (a + b) × h

כיצד למצוא גובה: הגובה הוא הקו הניצב לשני הבסיסים. בבגרות לרוב נתון ישירות.

דוגמה: a=6, b=4, h=3 ← שטח = ½×(6+4)×3 = ½×10×3 = 15

📖 פונקציה ליניארית — הסבר

m = שיפוע: כמה עולה הגרף לכל צעד ימינה. m>0 עולה, m<0 יורד, m=0 אופקי.

y = mx + b

נקודות חיתוך:

  • חיתוך עם ציר y: הנקודה (0, b) — נתון ישיר
  • חיתוך עם ציר x: כאשר y=0 ← x = −b/m

שיפוע בין שתי נקודות: m = (y₂−y₁) / (x₂−x₁)

📖 משוואה ריבועית — הסבר

המשוואה ax²+bx+c=0 נפתרת בנוסחת השורשים. לפני שמחשבים, מחשבים דיסקרימיננט:

Δ = b² − 4ac
  • Δ > 0 — שני שורשים שונים: x = (−b ± √Δ) / (2a)
  • Δ = 0 — שורש כפול: x = −b / (2a)
  • Δ < 0 — אין שורשים ממשיים

ראש הפרבולה: x = −b/(2a), y = f(x)

📖 גיאומטריה אנליטית — הסבר

נתונות שתי נקודות: P(x₁, y₁) ו-Q(x₂, y₂).

מרחק = √( (x₂−x₁)² + (y₂−y₁)² )
נקודת אמצע M = ( (x₁+x₂)/2 , (y₁+y₂)/2 )

שיפוע הישר PQ: m = (y₂−y₁) / (x₂−x₁)
