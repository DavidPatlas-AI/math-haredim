בעיות תערובת 🧪

סגסוגות, אחוזי מתכת, ערבוב נוזלים — טבלה אחת מסדרת הכול

מהי בעיית תערובת? הגדרה

בעיות תערובת עוסקות בערבוב של חומרים בריכוזים שונים. למשל: סגסוגת מתכת, תמיסת מלח, מיץ פירות.

כמות החומר הטהור = (אחוז / 100) × כמות התערובת
10 ק"ג סגסוגת עם 30% נחושת → נחושת טהורה: 0.3 × 10 = 3 ק"ג
5 ליטר תמיסת מלח 8% → מלח: 0.08 × 5 = 0.4 ליטר
💡 עיקרון המפתח
החומר הטהור נשמר. כשמערבבים שתי תערובות — כמות החומר הטהור בתערובת הסופית = סכום כמויות החומר הטהור בשתי התערובות המקוריות.

סוג 1: ערבוב שתי תערובות בסיסי

השיטה: בונים טבלה — לכל תערובת: כמות, אחוז, וכמות חומר טהור. סכום הכמויות = כמות סופית. סכום החומר הטהור = חומר טהור סופי.

תערובתכמות (ק"ג)אחוז (%)חומר טהור (ק"ג)
א'x20%0.2x
ב'y50%0.5y
סופיתx + y30%0.3(x+y)
כמה ק"ג מתערובת 20% ותערובת 50% צריך לערבב לקבלת 30 ק"ג ב-30%?
x + y = 30.   0.2x + 0.5y = 0.3·30 = 9.
פתרון: x=20, y=10.   20 ק"ג מ-20% + 10 ק"ג מ-50%.

סוג 2: הוספת חומר טהור חיזוק

מוסיפים חומר טהור (100%) לתערובת קיימת כדי להעלות את הריכוז.

ל-5 ק"ג סגסוגת 40% נחושת, מוסיפים נחושת טהורה. רוצים 60%. כמה להוסיף?
חומר טהור בהתחלה: 0.4·5 = 2.   מוסיפים x ק"ג טהור (100%).
סופי: (2 + x) / (5 + x) = 0.6 → 2+x = 3+0.6x → 0.4x = 1 → x = 2.5 ק"ג.

סוג 3: אידוי / הורדת חומר ריכוז

מאדים מים מתמיסה — כמות החומר הטהור נשארת, אבל הכמות הכוללת יורדת → הריכוז עולה.

10 ליטר תמיסת מלח 5%. מאדים מים. רוצים 8%. כמה מים התאדו?
מלח: 0.05·10 = 0.5 ליטר. אחרי אידוי x ליטר מים: 0.5/(10−x) = 0.08.
0.5 = 0.8 − 0.08x → 0.08x = 0.3 → x = 3.75 ליטר.

שיטת העבודה — טבלה מסודר

1. ציירו טבלה: עמודות — כמות, אחוז, חומר טהור.
2. מלאו את מה שידוע. סמנו נעלם = x.
3. רשמו משוואת שימור: חומר טהור לפני = חומר טהור אחרי.
4. פתרו, בדקו הגיון (כמות לא שלילית, אחוזים בין 0 ל-100).

טעויות נפוצות להיזהר

1. שוכחים להפוך אחוזים לשבר עשרוני. 30% = 0.3. תמיד לחלק ב-100.

2. מחברים אחוזים. 20% + 50% לא יוצא 70% כשמערבבים — זה תלוי בכמויות!

3. מבלבלים בין חומר טהור לכמות כוללת. חומר טהור = אחוז × כמות. לא להשוות כמות כוללת לחומר טהור.

4. שוכחים שב-100% זה 1.0. חומר טהור = 100% = 1.0. לא 100.
