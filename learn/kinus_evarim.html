כינוס איברים דומים

מהו פרמטר, ואיך מחברים ביטויים אלגבריים

מהו פרמטר? בסיס

פרמטר הוא אות שמציינת מספר. למשל a, b, x, y. עד שלא יודעים את ערכם, מתייחסים אליהם כאל מספרים.

a²a בריבוע = a·a
abמכפלת a ו-b (לפעמים בלי נקודת כפל)
a + bסכום a ו-b
(−a)המספר הנגדי ל-a
(−a) אינו בהכרח שלילי! אם a חיובי אז −a שלילי, אבל אם a שלילי אז −a דווקא חיובי.

מהם איברים דומים? הכלל

מותר לחבר ולחסר רק ביטויים זהים לחלוטין (איברים "דומים") — אותה אות ובאותה חזקה.

a + 3a + 5a − 2a = 7a
x² + 2x² = 3x²
3a + 2b + 2a + 3b = 5a + 5b
לא ניתן לחבר פרמטרים שאינם זהים: a ו-a² הם איברים שונים, וכך גם a ו-b.

דוגמה עם כמה סוגים

כשיש כמה סוגי איברים, מקבצים כל סוג בנפרד:

3a² − a + 3b − a² + 4a − b = 2a² + 3a + 2b
3x − 4xy − 5x + 11xy = −2x + 7xy
הטריק: סמנו כל סוג איבר בסימון משלו (קו תחתון, שני קווים...) כדי לא להתבלבל.

מה נחשב "איבר דומה"? הגדרה מדויקת

שני איברים דומים = אותה אות ואותה חזקה בדיוק. שינוי קטן → כבר לא דומים:

3a² ו-7a² → דומים ✓   (a בריבוע בשניהם)
3a² ו-7a → לא דומים ✗   (חזקות שונות)
5xy ו-3xy → דומים ✓   (אותן אותיות)
5xy ו-3x²y → לא דומים ✗   (חזקת x שונה)
כלל: כדי לכנס — חייב להיות זהה לחלוטין: אותן אותיות, אותן חזקות. המקדם (המספר שלפני) לא משנה.

כינוס עם כמה משתנים מסובך יותר

כשיש a, b, ו-ab — כל אחד נכנס בנפרד:

4a + 2b − ab + 3a − 5b + 2ab = 7a − 3b + ab
x² + 2xy − 3x + x² − xy + 5x = 2x² + xy + 2x
שימו לב: ab ו-a ו-b הם שלושה סוגים שונים. לא מחברים ביניהם.

כינוס עם איברים שליליים זהירות

כשיש מינוסים — שימו לב לסימן של כל איבר לפני הכינוס:

5a − (−3a) = 5a + 3a = 8a   (מינוס על מינוס = פלוס)
−4x − 2x + 7x = (−4 − 2 + 7)x = x
3b − 6b + b = (3 − 6 + 1)b = −2b
טעות נפוצה: כשיש מינוס לפני ביטוי בסוגריים, הוא הופך את כל הסימנים. −(3a − 2b) = −3a + 2b.

כינוס בצבעים 🎬

בחרו ביטוי ולחצו הפעל — איברים מאותו סוג יקבלו אותו צבע, ואז יתאחדו.
