חזקה של מספר שלילי

מתי שלילי בחזקה יוצא חיובי, ומתי שלילי

הכלל חשוב מאוד

חזקה = הכפלת המספר בעצמו כמה פעמים. סימן התוצאה נקבע לפי מספר הסימנים השליליים:

(−)זוגי = + חיובי
(−)אי-זוגי = − שלילי
(−2)⁴ = 16  (חזקה זוגית → חיובי)
(−2)⁵ = −32  (חזקה אי-זוגית → שלילי)
(−3)⁴ = (−3)·(−3)·(−3)·(−3) = 81

הטעות הנפוצה: סוגריים! להיזהר

ההבדל בין (−x)² לבין −x² הוא קריטי:

(−2)² = 4
החזקה על המספר והסימן → חיובי
−2² = −4
החזקה רק על המספר, המינוס נשאר → שלילי
כשאין סוגריים, החזקה מתייחסת רק למספר ולא לסימן:
−2⁴ = −16   אבל   (−2)⁴ = 16
(−3)⁴ = 81  לעומת  −3⁴ = −81
(−3)³ = −27  וגם  −3³ = −27  (כאן זהה!)

פתרון שלב-שלב 🎬

בחרו תרגיל — שימו לב להבדל בין יש סוגריים לאין סוגריים.
