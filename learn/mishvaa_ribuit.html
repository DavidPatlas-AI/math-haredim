משוואה ריבועית

פתרון משוואות עם x² — הנוסחה שצריך לשנן

מה זו משוואה ריבועית? בסיס

משוואה ריבועית היא משוואה שבה ה-x מופיע בחזקה 2. הצורה הכללית:

ax² + bx + c = 0

שלושת המספרים a, b, c הם המקדמים. a חייב להיות שונה מאפס (אחרת אין x²).

2x² + 3x − 5 = 0 → a=2, b=3, c=−5
x² − 7x + 12 = 0 → a=1, b=−7, c=12
3x² + 6 = 0 → a=3, b=0, c=6

נוסחת השורשים הנוסחה המרכזית

זו הנוסחה הכי חשובה בבגרות — חייבים לשנן בעל-פה!

x1,2 = −b ± √(b² − 4ac)2a

הסימן ± אומר שיש שני פתרונות: אחד עם פלוס, אחד עם מינוס.

הדיסקרימיננטה קובעת כמה פתרונות יהיו

הביטוי שבתוך השורש — b² − 4ac — נקרא דיסקרימיננטה (סימן: Δ). הוא קובע כמה פתרונות יש למשוואה:

Δ > 0
שני פתרונות שונים
Δ = 0
פתרון אחד בלבד
Δ < 0
אין פתרון (במספרים ממשיים)
💡 למה זה?
הדיסקרימיננטה אומרת כמה פעמים הפרבולה חוצה את ציר ה-x:
Δ > 0 = שתי חציות ← שני שורשים
Δ = 0 = נגיעה אחת בציר ← שורש יחיד
Δ < 0 = לא מגיעה לציר ← אין פתרון

אי אפשר להוציא שורש ממספר שלילי. בבגרות, אם יוצא √שלילי — כותבים "Δ < 0, אין פתרון ממשי".

צעדי פתרון תהליך

1. מסדרים את המשוואה לצורה ax² + bx + c = 0
2. מזהים את a, b, c
3. מחשבים את הדיסקרימיננטה: Δ = b² − 4ac
4. מציבים בנוסחה ומקבלים את x₁ ו-x₂
5. בודקים ע"י הצבה

דוגמה מלאה

פתרו: x² − 5x + 6 = 0

a=1, b=−5, c=6
Δ = (−5)² − 4·1·6 = 25 − 24 = 1
√Δ = √1 = 1
x₁ = (5 + 1) / 2 = 3
x₂ = (5 − 1) / 2 = 2
בדיקה: 3² − 5·3 + 6 = 9 − 15 + 6 = 0 ✓   2² − 5·2 + 6 = 4 − 10 + 6 = 0 ✓

טעויות נפוצות להיזהר

1. שימו לב לסימן של b! אם המשוואה x²−5x+6, אז b=−5 (לא 5). וב-(−b) זה כבר ייצא +5.

2. אל תשכחו לחלק ב-2a את כל המונה — לא רק את −b.

3. √שלילי = אין פתרון. אל תכתבו "0" — כתבו "אין פתרון ממשי".

כשΔ שלילי — אין פתרון מקרה 3

כשהדיסקרימיננטה שלילית — אי אפשר להוציא שורש. המשוואה אין לה פתרון במספרים ממשיים:

x² + 2x + 5 = 0 → a=1, b=2, c=5
Δ = 4 − 4·1·5 = 4 − 20 = −16
Δ < 0 → אין פתרון ממשי
בבגרות: כשמגיעים לשורש שלילי — כותבים "Δ < 0, אין פתרון במספרים ממשיים" וממשיכים הלאה. זה תשובה מלאה.

כשΔ = 0 — פתרון יחיד מקרה 2

כש-Δ = 0 — הביטוי ±√0 = 0. שני הפתרונות זהים — מקבלים פתרון אחד בלבד:

x² − 6x + 9 = 0 → a=1, b=−6, c=9
Δ = 36 − 36 = 0
x = 6/2 = 3   (פתרון יחיד)
בדיקה: 3² − 6·3 + 9 = 9 − 18 + 9 = 0 ✓
שימו לב: x² − 6x + 9 = (x−3)² — זה ריבוע סכום. כשיש ריבוע שלם — תמיד Δ=0 ופתרון יחיד.

נוסחאות ויאטה — סכום ומכפלת השורשים קיצור חשוב

אם x₁ ו-x₂ הם שני שורשי המשוואה ax² + bx + c = 0, אז בלי לחשב בכלל:

x₁ + x₂ = −b/a     x₁ · x₂ = c/a
x² − 5x + 6 = 0 → סכום: −(−5)/1 = 5   מכפלת: 6/1 = 6   (ואכן: 2+3=5, 2·3=6 ✓)
2x² + 3x − 2 = 0 → סכום: −3/2   מכפלת: −2/2 = −1
מתי משתמשים? בעיקר לשתי מטרות:
1. לבדוק תשובה מהר — אחרי שמצאתם שורשים, בדקו שהסכום והמכפלה נכונים.
2. כשנותנים בשאלה מידע על הסכום או המכפלה — אפשר לפרוק ישירות בלי נוסחת שורשים.
